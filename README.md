## Requirement
### install lazygit
* Windows 
(Scoop)
```shell
# Add the extras bucket
scoop bucket add extras

# Install lazygit
scoop install lazygit
```
(Chocolatey)
```shell
choco install lazygit
```
(Winget)
```shell
winget install -e --id=JesseDuffield.lazygit
```

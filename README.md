## Requirement
### install lazygit
* Windows
1. In Scoop
```shell
# Add the extras bucket
scoop bucket add extras

# Install lazygit
scoop install lazygit
```
2.Chocolatey
```shell
choco install lazygit
```
3.Winget
```shell
winget install -e --id=JesseDuffield.lazygit
```
* Termux
```shell
apt install lazygit
```
* Ubuntu/Debian
```shell
sudo apt install lazygit
```
